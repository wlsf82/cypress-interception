# Cypress Interception

👋 Welcome!

It's great to have you here. I'm sure you're going to love this course. ❤️

## What you'll learn

During the Cypress Interception course, you'll learn:

- How to create 100% flakiness-free tests
- How to interact with the application as close as possible to how a real user would
- How to wait for an element to be visible before interacting with it
- How to simulate keyboard usage instead of clicking a button in search features
- How to wait for HTTP requests before moving forward with the tests
- How to organize different tests into their specific contexts
- How to read the browser's `localStorage`
- How to make element selectors more specific
- How to simplify repetitive commands
- How to intercept and mock an external API to test the frontend independently of the backend
- How to simulate server and network errors
- How to simulate slowness in an API call
- How to customize the default test report to match your style 😉
